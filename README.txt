DEPENDENCIES:
- Numpy
- Pandas
- Random

RUNNING:
- assuming all dependencies are installed the script can be run as any python script
- {/path/to/python} main.py

INPUT:
- input files are coded into the file itself
- can be changed by changing the names of files in the "categories" array
- input file contain each word and then 300 features
- file is preset to reproduce output that is shown in the pdf, this is because the choice variable is fixed to 1
- can be changed to any value within the length of the array or to random by setting choice=None

OUTPUT:
- all numeric output is printed to the console
- Plots are saved to the current directory
